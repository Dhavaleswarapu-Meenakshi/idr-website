# Institute of Digital Risk – Homepage

This project is a responsive single-page website created as part of an assignment.

## Features
- Responsive layout
- Sticky navigation bar
- Smooth scrolling
- Semantic HTML5 structure
- Service pillar cards
- Contact form

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Sections
- Hero section
- About IDR
- Service pillars
- Community
- Contact form

## Branding
The logo uses a geometric cube to represent structured digital systems and digital risk frameworks. The color palette uses orange, black, and white to convey innovation, security, and clarity.
